# Haan Handmade Boutique Website

Welcome to the **Haan Handmade** website project! This is a modern, elegant, and mobile-friendly storefront designed for a handcrafted boutique business. 

## 🛠️ Languages & Technologies Used

This project was built using standard web technologies with a focus on high performance and maximum customizability without relying on heavy frameworks.

- **HTML5**: Used for semantic structure and layout.
- **CSS3 (Vanilla)**: Used for all styling, including responsive layouts (Grid/Flexbox), custom color variables, and keyframe animations.
- **JavaScript (Vanilla)**: Used for interactivity such as the mobile navigation toggle, smooth scroll animations (via `IntersectionObserver`), and simulating form submissions.
- **Vite**: A fast build tool and development server used to run the project locally with hot-reloading.

## 📂 File Structure & Where to Find Code

The project files are organized as follows:

```text

```

## 🚀 How to Run Locally

If you want to run the website on your local machine, follow these steps:

1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Open a terminal in the project folder and run:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to `http://localhost:5173/` (or the URL provided in the terminal).
